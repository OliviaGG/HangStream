Deployment checklist

Set the OAuth env vars in your host, make sure the Google client includes both redirect URIs, and verify the service is exposed on the platform URL before sharing it publicly.

Google redirect URIs

Register both callback paths in Google Cloud Console.

https://YOUR_DOMAIN/oauth/google/callback https://YOUR_DOMAIN/auth/google/callback

Runtime notes

HangStream reads `PORT` from the environment, so it can run on Cloud Run, DigitalOcean App Platform, or a container host without any code changes.
